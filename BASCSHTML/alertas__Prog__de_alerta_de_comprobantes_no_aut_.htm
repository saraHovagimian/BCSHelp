Autorizaciones pendientes 

El sistema brinda la posibilidad de programar procesos y acciones del sistema para que se ejecuten en forma automática a determinados horarios y con determinada periodicidad.

Los procedimientos consisten en el envío de correo electrónico a uno o más usuarios ante la ocurrencia de ciertas circunstancias en el sistema. El sistema permite configurar quiénes son los destinatarios de estos mensajes y ante qué circunstancias se desencadenan los alertas, según las necesidades de cada caso.

Aquí se puede programar cuándo y cómo se desea que el sistema avise que un comprobante debe ser autorizado a los usuarios que correspondan.

Según la tabla de autorizaciones que se utilice para autorizar un determinado comprobante, si se indicó que se debe avisar a los usuarios cada vez que deban autorizar, mediante este proceso se podrá configurar cuándo se enviarán los mensajes. La dirección de correo electrónico de cada usuario involucrado en la autorización podrá indicarse en la ficha Datos generales del punto Usuarios.

Se enviarán primero los alertas a los usuarios de menor nivel que deban autorizar, una vez que estos autorizan, se enviarán los alertas a los usuarios del nivel siguiente y así sucesivamente.

Se podrá programar la periodicidad con que se enviarán los alertas y la frecuencia diaria, se podrá configurar para que se envíen los mensajes siempre un día y a una determinada hora o se pueden indicar períodos de días, semanas o meses pasados los cuales se enviarán.

La programación periódica de alertas sólo podrá ser realizada por el administrador del sistema y podrá modificarse esta programación todas las veces que sea necesario.

Periódico

A través de esta selección es posible programar la tarea para que se realice en base a un período determinado (días, semanas, meses).

Sucede

Diariamente

Semanalmente

Mensualmente

En el primer caso es posible parametrizar el sistema para que el proceso se ejecute cada n cantidad de días.

Si se selecciona la opción semanalmente se puede parametrizar para que el proceso se ejecute cada n semanas o en determinados días de la semana.

Si se selecciona la opción mensualmente se parametriza para que el proceso se ejecute cada n meses o determinado día de cada mes, o el primer (segundo, tercero, cuarto o útimo) día, por ejemplo martes, de cada mes.

Frecuencia diaria

En base a esta selección se programa el proceso para que se efectúe diariamente. Las posibilidades son:

Sucede una vez a las:

Sucede cada:

n horas o minutos, comenzando a las hora determinada finalizando a las hora determinada. (Por ejemplo: de 8:00:00 a 18:00:00 hs.).

Una vez

Si se indica esta opción el proceso se ejecuta un sola vez el día y hora indicadas.

Agregar salida al historial de la tarea

Si se tilda esta opción se emite un graban los pasos del proceso que se ejecuta en el SQL.
