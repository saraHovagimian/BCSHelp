Emisión de informes

Este informe permite listar los informes que se han impreso o se han emitido por pantalla, Excel o tabla, mostrando qué usuario lo envió a imprimir o a emitir, desde qué puesto de trabajo y con qué opciones lo hizo.

Para emitir el listado primero se ingresan los datos en las distintas fichas contenidas en la tarea, de manera tal de filtrar la información de acuerdo a las necesidades. Luego se debe presionar el ícono (botón) de la barra de herramientas superior que corresponda al tipo de emisión requerida. El uso de estos botones se detalla en el link Emisión disponible en un párrafo posterior.

Al momento de imprimir un informe (tanto por pantalla como por papel) se puede utilizar el formato standard del sistema, o indicar un formato personalizado de impresión. Este último se indica en el campo homónimo que, en general, se encuentra en la zona inferior de la primer solapa y su funcionalidad está detallada a continuación en el link Emisión.

Emisión

Los datos que a ingresar son:

Ficha OpcionesFicha Opciones

Usuario

Nombre del usuario que emitió el informe.

Puesto

Puesto desde donde se emitió el informe.

Período a considerar

Desde hora

Este campo sólo se activa si se seleccionó la opción fecha de Registración.

Aquí se informará la hora a partir de la cual se desean incluir los comprobantes correspondientes a la fecha de registración indicada en el campo anterior. El sistema ofrecerá por defecto la hora 00:00:00.

Hasta hora

Este campo sólo se activa si se seleccionó la opción fecha de Registración.

Hora hasta la cual se desean incluir los comprobantes correspondientes a la fecha de registración indicada en el campo anterior. El sistema ofrecerá por defecto la hora 23:59:59.

Si no se indica ninguna hora, se incluyen los comprobantes ingresados hasta el final.

Ordenamiento

El ordenamiento de este informe puede ser:

Usuario

Indicando esta opción este listado se emite ordenado por usuario.

Fecha de impresión

Indicando esta opción este listado se emite ordenado por fecha de registración de la impresión.

Informe

Indicando esta opción este listado se emite ordenado por nombre de informe.

Destino

Es posible filtrar este listado por destino. Las opciones son:

Pantalla

Impresora

Excel

Tabla

Todos

Informes con duración mayor a segundos

Completando este campo es posible filtrar los informes con duración mayor a la cantidad de segundos que se desee.
