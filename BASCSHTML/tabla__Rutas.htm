Rutas 

Aquí se administra toda la información relativa a las rutas o circuitos de entrega con los que opera la empresa.

Muchas empresas establecen distintas áreas de entrega de mercadería en función a la ubicación geográfica y domicilios de entrega de los clientes.

Existen procesos automáticos que permiten generar simultáneamente las facturas que corresponden a un determinado circuito de entrega.

El sistema prevé asociar, a cada cliente que se incorpore al sistema, una serie de rutas. Cada ruta representa un circuito de entrega que incluye a ese cliente en su itinerario.

Luego, los procesos que permiten facturar los pedidos que aún se encuentran pendientes, permitirán seleccionar solamente aquellos que corresponden a los clientes que pertenecen a determinada ruta.

Los datos que deben ingresarse son los siguientes:

Código Código

Código de hasta tres caracteres que identifica a la ruta.

Como ejemplos podemos mencionar, R03, 85, etc.

DescripciónDescripción

Descripción que se le da a la ruta que se está definiendo. Se dispone de hasta veinticinco caracteres para el ingreso.

Lunes-Martes-Miércoles-Jueves-Sábado-DomingoLunes-Martes-Miércoles-Jueves-Sábado-Domingo

Permite indicar, mediante tildes, los días de la semana en los que se debe realizar la ruta, es decir, si se debe realizar los lunes y/o martes y/o miércoles y/o jueves y/o viernes y/o sábados y/o domingos.

Tabla de ClientesTabla de Clientes

Tabla que contiene la lista de clientes que pertenecen al circuito de entrega que se está definiendo.

Esta tabla puede ingresarse y modificarse desde este punto, es decir, definiendo qué clientes se encuentran en cada ruta; o bien, desde el ingreso de clientes, definiendo en qué rutas se encuentra cada cliente.

Número de orden

Número que indica el orden que tiene el cliente dentro de la ruta.

Cliente

Código del cliente que se asocia a la ruta. Debe ingresarse un código de cliente ya definido en el sistema.

Domicilio

Domicilio del cliente indicado. Este dato se completará automáticamente al ingresar el código del cliente.

Localidad

Localidad del cliente indicado. Este dato se completará automáticamente al ingresar el código del cliente.
