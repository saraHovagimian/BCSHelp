Prefijo de comprobantes externos

En los ingresos de comprobantes externos, al ingresar el prefijo del número interno, se aplicará el formato para prefijos indicado en la empresa para la cual se ingresa el comprobante.

Al ingresar el prefijo del número externo se aplicará el formato que se esté utilizando como default para prefijos externos. Ese formato “Por defecto” puede modificarse por otro, haciendo clic en el botón que se muestra a la derecha de campo "Número externo".

Al hacer clic aparece una ventana mostrando los prefijos ingresados en el ABM de formatos para prefijo, pudiendo elegir un prefijo ocasional, es decir, para ser utilizado sólo en el comprobante que se está registrando.

Se puede seleccionar una fila, presionar el botón “Por defecto” y quedará elegido el nuevo formato como formato "por defecto".

En cambio, si se selecciona un formato y se pulsa “Aceptar”, se cambia el formato para el comprobante que estamos ingresando, pero no se cambia el formato por defecto.

En la figura puede verse que se dispone de la opción “S/F Sin formato”. Al seleccionarla, no se validará el formato de prefijo  a ingresar, sino que se podrá ingresar un prefijo de hasta 8 caracteres alfanuméricos. Al ingresarlo,no se les aplicará ningún formato, sino que se tomará como prefijo textualmente lo que se ingrese.

Si por ejemplo, se selecciona el formato “000-000” en el comprobante de la figura y se indica "Por defecto", al ingresar cualquier otro comprobante con número externo, se ofrecerá el formato de prefijo externo “000-000”, pudiendo cambiarlo haciendo clic en el botón disponible a la derecha de "Número externo".

Si se quiere ingresar un comprobante con un determinado prefijo externo, si el formato de prefijo que se está utilizando no corresponde al del prefijo a ingresar, se deberá cambiar ese formato ya que el prefijo externo que se indique se validará usando el último formato utilizado. Si, por ejemplo, se están ingresando facturas de compra  utilizando el formato “000-000”, y se requiere ingresar una compra a un proveedor  que utiliza el formato “A000”, si se indica el prefijo A001 sin cambiar el formato, aparecerá una ventana indicando que el formato no es correcto. En este caso, si se hubiera seleccionado previamente  el formato “S/F Sin formato”, no se hubiesen presentado inconvenientes al cambiar el formato del prefijo.

Si se ingresan comprobantes utilizando modelos, si se utiliza un modelo que tenga guardado un prefijo externo con un formato que no es el que se está utilizando al momento del ingreso, el modelo se recuperará sin problemas conservando el formato de prefijo externo del mismo. Si luego se ingresa otro comprobante "Sin modelo", o con otro modelo que no tiene guardado un prefijo externo, el default del formato vuelve a ser el que esté indicado como prefijo "Por defecto" que se estaba utilizando antes de cargar el modelo anterior.
