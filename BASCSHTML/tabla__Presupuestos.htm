Definición de presupuestos

El sistema permite definir los presupuestos de la empresa para un determinado período de tiempo.

Para cada presupuesto definido en este punto podrán ingresarse luego las componentes del mismo, en donde se informan los importes presupuestados para las diferentes cuentas presupuestarias que conforman el presupuesto.

Se pueden definir todos los presupuestos que se sean necesarios.

La información a ingresar al definir presupuestos es la siguiente:

CódigoCódigo

Código de hasta tres caracteres que identifica al presupuesto y será utilizado para individualizar al mismo en todo el sistema.

DescripciónDescripción

Denominación del presupuesto. Se dispone de hasta veinticinco caracteres para el ingreso.

Por ejemplo, Presupuesto año 1998, Primer bimestre 1999, etc.

Esta información se visualizará en todos los informes en los que intervenga el presupuesto que se está definiendo.

MonedaMoneda

Número de moneda del presupuesto.

Debe corresponder al número de una tabla de tipo Cotización.

Si no se indica ningún valor se asume que el presupuesto se ingresa en moneda local. Si se ingresa una moneda, los valores se ingresarán al presupuesto expresados en esa moneda.

Los informes de presupuestos mostrarán los valores expresados en la moneda que aquí se indique.

EstadoEstado

Indicación del estado del presupuesto que se está definiendo. El Estado del presupuesto determina el tipo de operaciones que se pueden efectuar con el mismo.

Los valores posibles de seleccionar son:

Abierto

Cerrado

Si se indica estado Abierto, podrán definirse y modificarse las componentes del presupuesto.

Si se indica estado Cerrado, sólo podrán consultarse las componentes del presupuesto ingresadas y no podrán ingresarse nuevas componentes.

PeríodoPeríodo

Período para el que se define el presupuesto.

Desde

Fecha desde la cual es válido el presupuesto que se está definiendo.

Al ingresar las componentes del presupuesto, se podrá indicar para cada una un período de tiempo distinto a este, siempre que esté comprendido dentro del rango de fechas del presupuesto.

Hasta

Fecha hasta la cual es válido el presupuesto que se está definiendo.

Al ingresar las componentes del presupuesto, se podrá indicar para cada una un período de tiempo distinto a este, siempre que esté comprendido dentro del rango de fechas del presupuesto.
