Tipos de comprobantes

Todas las tareas del sistema apuntan y están relacionadas, fundamentalmente, al registro de Comprobantes.

Se prevén comprobantes para registrar todas las operaciones habituales que pueden tener lugar en la organización.

El sistema ya tiene incorporados una gran cantidad de tipos de comprobantes que permiten registrar todas las operaciones del sistema sin necesidad de efectuar la definición previamente.

En este punto se pueden visualizar cuáles son los distintos tipos de comprobantes que se utilizarán, y de ser necesario, podrán modificarse algunos de sus datos.

Los datos que deben ingresarse son los siguientes:

OrdenamientoOrdenamiento

A la derecha del rótulo Ordenamiento, se indica el criterio por el cual están ordenadas las filas y se ofrece un enlace para indicar otros criterios.

Por defecto, la tabla se presenta ordenada por talonario. Al hacer clic en el enlace se habilita una ventana que permite definir el ordenamiento que se le desea dar a los elementos en la grilla. Es posible ordenar por todas las columnas de la misma.

CódigoCódigo

Código de dos caracteres que identifica al tipo de comprobante. Aquí se deberá ingresar un código de los ya definidos en el sistema.

Como ejemplos podemos mencionar, FA, RQ, OP, etc.

NombreNombre

Nombre del comprobante. Se dispone de hasta treinta caracteres para el ingreso.

Por ejemplo, Factura de Venta A, Requerimiento de Compra, Orden de Pago, etc.

Se exhibe la descripción del comprobante utilizada por el sistema y, de ser necesario, podrá modificarse por el usuario.

Esta información figurará en todas las tareas del sistema que permitan seleccionar comprobantes de una lista dada.

AbreviadoAbreviado

Nombre abreviado del comprobante. Se dispone de hasta cinco caracteres para el ingreso.

Por ejemplo, FAC A, REQ, O.P., etc.

Se exhibe la descripción abreviada del comprobante utilizada por el sistema y, de ser necesario, podrá modificarse por el usuario.

Este dato figurará en los informes del sistema en los que se haga referencia a comprobantes del tipo indicado.

Ver todas las vigenciasVer todas las vigencias

Por defecto el programa muestra para cada tipo de comprobante los talonarios que permiten su ingreso sólo para la vigencia actual. Si se indica esta opción se muestran todas las vigencias de los talonarios detallados para el tipo de comprobante.

Tabla de TalonariosTabla de Talonarios

Se exhibe en esta tabla toda la información asociada a los talonarios habilitados para operar con el tipo de comprobante indicado.

Estos datos vinculados a los talonarios no podrán modificarse desde este punto del sistema, sino desde la definición de talonarios. Se muestra aquí a los efectos de brindar mayor claridad y facilidad para acceder a la información.

Se muestran todas las vigencias del talonario o sólo la actual, según si se elige o no ver todas las vigencias.

Talonario

Código del talonario habilitado para operar con el tipo de comprobante indicado.

Descripción

Denominación del talonario habilitado para operar con el tipo de comprobante indicado.

Prefijo

Prefijo del talonario habilitado para operar con el tipo de comprobante indicado.

Numeración

Tipo de numeración del talonario habilitado para operar con el tipo de comprobante indicado.

Vigencia Desde

Fecha desde de la vigencia de talonarios que se detalla.

Vigencia Hasta

Fecha hasta de la vigencia de talonarios que se detalla.

Número Inicial

Número inicial de la vigencia de talonarios que se detalla.

Número Final

Número final de la vigencia de talonarios que se detalla.

Próximo

Próximo número de comprobante de la vigencia del talonario habilitado para operar con el tipo de comprobante indicado.
