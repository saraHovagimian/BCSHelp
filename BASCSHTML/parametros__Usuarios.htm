Usuarios

Esta tarea permite proporcionar al sistema una serie de datos asociados al usuario que está utilizando el sistema. Estos datos contribuyen a facilitar y automatizar determinados procesos del sistema vinculados con el ingreso de comprobantes y con la emisión de informes, cuando se trabaja con múltiples empresas y sucursales .

El sistema ya tiene incorporados algunos valores de parámetros, pero los mismos pueden modificarse para adecuarse a las necesidades de cada caso.

Los parámetros de cada usuario pueden modificarse mientras el mismo está operando con el sistema y la actualización de las opciones que se indiquen se realizará en forma inmediata sin necesidad de salir del sistema y volver a entrar.

Usuario

Al ingresar en esta ventana se muestra el usuario para el cual se modificarán los parámetros y se podrán configurar los siguientes parámetros del usuario.

Operar con

Empresa

Empresa a la que ingresa el usuario al operar con el sistema. Esta empresa puede modificarse desde este punto o desde el punto Usuarios.

Es un número de hasta dos dígitos que debe corresponder a una empresa definida en el sistema.

En el punto Usuarios se pueden habilitar las empresas a utilizar por cada usuario o grupo de usuarios del sistema, en el caso en que se desee acotar el grupo de empresas permitidas para cada usuario.

Si el usuario o el grupo al cual pertenece el usuario, tiene empresas habilitadas para ingresar comprobantes, sólo se permitirá el ingreso de una empresa habilitada. Si el usuario no tiene empresas habilitadas, se podrá indicar cualquier empresa definida en el sistema.

Esta empresa puede ser modificada por otra cada vez que se necesite, y sólo afecta a las nuevas transacciones que registre el usuario, y no a las registradas por el mismo en otras empresas.

Sucursal

Sucursal de la empresa a la que ingresa el usuario al operar con el sistema. Esta sucursal puede modificarse desde este punto o desde el punto Usuarios.

Es un número de hasta dos dígitos que debe corresponder a una sucursal definida en el sistema para la empresa indicada en el dato anterior.

En el punto Usuarios se pueden habilitar las sucursales de las empresas a utilizar por cada usuario o grupo de usuarios del sistema, en el caso en que se desee acotar el grupo de empresas y sucursales permitidas para cada usuario.

Si el usuario o el grupo al cual pertenece el usuario, tiene empresas y sucursales habilitadas para ingresar comprobantes, sólo se permitirá el ingreso de una sucursal habilitada. Si el usuario no tiene empresas habilitadas, se podrá indicar cualquier sucursal definida en el sistema para la empresa indicada.

También existe la posibilidad de no indicar una sucursal en este campo con lo cual, el usuario podrá ingresar comprobantes y realizar procesos en las distintas sucursales de su empresa , sin tener que cambiar de sucursal cada vez.

En el caso de no tener sucursal, la sucursal de cada comprobante que el usuario registre se determinará por la sucursal asociada al talonario, al depósito o a la caja que se utilicen en el comprobante (dependiendo del tipo de comprobante que se ingresa).

Hay algunos procesos que necesitan utilizar una sucursal determinada y si el usuario no tiene sucursal esto será informado al ejecutar esos procesos.

Hay tareas como por ejemplo, el cierre operativo de subdiarios, la renumeración de asientos, los asientos de cierre y los de ajuste, que involucran a todas las sucursales y exigen que el usuario que los ejecuta tenga asociada la sucursal correspondiente a la casa central. Esto también será informado en caso de ser necesario.

La sucursal del usuario puede ser modificada por otra cada vez que se necesite, y sólo afecta a las nuevas transacciones que registre el usuario, y no a las registradas por el mismo en otras sucursales.

Informes multiempresa

El usuario puede emitir informes consolidando empresas y sucursales, o listar la información de una determinada empresa o sucursal.

Para esto se permite aquí seleccionar las empresas y sucursales cuya información se consolidará cuando el usuario emita los informes multiempresa del sistema.

Esta información es opcional y las empresas y sucursales que aquí se indiquen se ofrecerán como default al emitir los informes multiempresa, pudiendo modificar las mismas al emitir cada informe.

En el punto Usuarios se pueden habilitar las empresas y sucursales a utilizar por cada usuario o grupo de usuarios del sistema, en el caso en que se desee acotar el grupo de empresas y sucursales permitidas para cada usuario.

Si el usuario o el grupo al cual pertenece el usuario, tiene empresas habilitadas para emitir informes, sólo se permitirá consolidar información de empresas y sucursales habilitadas. Si el usuario no tiene empresas habilitadas para informes, se podrá consolidar la información de cualquier empresa y sucursal definida en el sistema.

Seleccionar sólo empresa corriente

Si se activa esta opción, en la solapa "Empresa" de los informes multiempresa, sólo aparece seleccionada la empresa del usuario la que se informó en el campo "Operar con".

Esto es así siempre y cuando en el informe multiempresa del que se trate no se haya definido un modelo o, en caso de haberlo definido, no se hayan incluido las empresas en él. Si el informe tiene definido un modelo que incluye empresas, aparecerán seleccionadas las empresas u sucursales correspondientes al mismo.

Descripción

Denominación del grupo de empresas y sucursales utilizadas por el usuario para emitir informes. Esta denominación se visualizará en todos los informes que emita el usuario.

Esta información es opcional y se ofrecerá como default cuando el usuario emita los informes multiempresa, pudiendo modificar la misma al emitir cada informe.

Puede ingresarse el texto en forma manual o puede pulsarse el botón Completar con empresas que completa automáticamente el dato con los nombres de todas las empresas seleccionadas para emitir informes.

Tabla de empresas

Esta tabla muestra la empresa lista de empresas posibles de seleccionar para emitir los informes del sistema.

Las empresas no se ingresan manualmente en la tabla, sino que la misma se completa automáticamente según las empresas que el usuario ya tenga seleccionadas y las empresas habilitadas para el usuario en el punto Usuarios.

Si el usuario tiene todas las empresas habilitadas, se completará la lista con todas las empresas existentes. En ambos casos, las empresas ya seleccionadas para el usuario, se mostrarán resaltadas en color rojo.

Empresa

Código de la empresa cuya información podrá ser visualizada por el usuario al emitir los informes del sistema.

Las empresas ya seleccionadas por el usuario se mostrarán resaltadas en color rojo y las no seleccionadas podrán seleccionarse haciendo doble click en las filas correspondientes.

Cada vez que se marca o se desmarca una empresa se produce el mismo efecto en las sucursales de la misma.

Al seleccionar una empresa, se seleccionan automáticamente todas las sucursales habilitadas de la misma, pudiendo luego desmarcar las sucursales de la empresa que no se deseen incluir en los informes.

Al desmarcar una empresa, se desmarcan automáticamente todas las sucursales habilitadas de la misma, y no podrá ser posible seleccionar a ninguna sucursal de la empresa.

Pulsando el botón Invertir selección se seleccionan automáticamente todas las empresas de la lista que no estaban marcadas y todas las que lo estaban quedan no seleccionadas.

Descripción

Descripción correspondiente a la empresa cuyo código se exhibe en el dato anterior.

Tabla de sucursales

Esta tabla muestra la lista de sucursales de la empresa cada empresa posibles de seleccionar para emitir los informes del sistema.

Las sucursales no se ingresan manualmente en la tabla, sino que la misma se completa automáticamente según las empresas y sucursales que el usuario ya tenga seleccionadas y las empresas y sucursales habilitadas para el usuario en el punto Usuarios.

Si el usuario tiene todas las empresas habilitadas, se completará la lista con todas las sucursales existentes de cada empresa. En ambos casos, las sucursales ya seleccionadas para el usuario, se mostrarán resaltadas en color rojo.

Sucursal

Código de la sucursal cuya información podrá ser visualizada por el usuario al emitir los informes del sistema.

Las sucursales ya seleccionadas por el usuario se mostrarán resaltadas en color rojo y las no seleccionadas podrán seleccionarse haciendo doble click en las filas correspondientes.

Pulsando el botón Invertir selección se seleccionan automáticamente todas las sucursales de la lista que no estaban marcadas y todas las que lo estaban quedan no seleccionadas.

Descripción

Descripción correspondiente a la sucursal cuyo código se exhibe en el dato anterior.
