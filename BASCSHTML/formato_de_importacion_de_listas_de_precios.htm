Formato de importación de listas de precios

En el campo Formato es posible indicar el formato correspondiente al archivo a importar o presionar la tecla de F4 para dar de alta un formato o modificar uno ya existente.

El archivo deberá tener como campos los códigos de ítems. Precios y, si corresponde los códigos de color y de medida.

Si la lista es de proveedor está previsto que los códigos sean los códigos propios que se hayan informado para dicho proveedor.

Al acceder al alta de un formato se abre una ventana donde se debe informar:

FormatoFormato

Código del formato.

DescripciónDescripción

Nombre del formato del formato.

Luego se debe dar información de parametrización destinada a describir como están organizados los datos en el archivo a importar.

ConfiguraciónConfiguración

Se debe indicar:

Hay líneas de títulos

Tildar si la primera línea del archivo corresponde a títulos.

Contiene datos de color y medida

Tildar en caso que el archivo incluya campos con los códigos de los colores y las medidas de los ítems.

si la primera línea del archivo corresponde a títulos.

FormatoFormato

Se debe seleccionar alguna de las dos opciones que se presentan:

C.S.V. (valores separados por comas)

Se debe esta opción si los datos en el archivo están separados por comas. Al hacerlo, en la tabla de la derecha se indicarán los datos del archivo y el orden en que se presentan.

Los datos se deben seleccionar en la columna “Campo” y el orden en la columna “Posición”. Los posibles datos previstos son: Ítem, Precios, Medida y Color (estos dos últimos solo se presentan si se indicó que el archivo contiene datos de color y medida). En “Posición”, se indicará 1 para el primer dato del registro, dos para el segundo y así sucesivamente.

Posición fija

Esta opción se seleccionará si los datos en el archivo ocupan posiciones fijas.

Al seleccionar la opción se habilita el campo “Longitud del registro” donde se deberá indicar dicha información.

En este caso, en la tabla de la derecha, se deberá informar para cada dato, la columna del registro donde comienza y su longitud.
