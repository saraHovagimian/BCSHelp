Informe de novedades

Este informe permite ver las novedades ingresadas, para ser facturadas a los clientes con los que se opera.

Pueden listarse las novedades de clientes, las novedades de los atributos de los clientes o todas. Esta última opción agrupa todas las novedades de cada cliente tal como se hace en el momento de facturar las novedades del cliente.

Se pueden imprimir las novedades de facturación por rangos de clientes y se puede seleccionar el tipo de procesamiento de las novedades a listar.

Se muestra para cada cliente o para cada atributo y valor de atributo de clientes, el detalle de las novedades asociadas con código de ítem, cantidad, precio, bonificación, importe, centros de apropiación, prioridad de la novedad y tipo de proceso para el que es válida la misma.

Es posible, además, incluir en la impresión del informe una carátula en la que se detallan todas las opciones seleccionadas por el usuario al solicitarlo.

Para emitir el listado primero se ingresan los datos en las distintas fichas contenidas en la tarea, de manera tal de filtrar la información de acuerdo a las necesidades. Luego se debe presionar el ícono (botón) de la barra de herramientas superior que corresponda al tipo de emisión requerida. El uso de estos botones se detalla en el link Emisión disponible en un párrafo posterior.

Al momento de imprimir un informe (tanto por pantalla como por papel) se puede utilizar el formato standard del sistema, o indicar un formato personalizado de impresión. Este último se indica en el campo homónimo que, en general, se encuentra en la zona inferior de la primer solapa y su funcionalidad está detallada a continuación en el link Emisión.

Emisión

Los datos solicitados para la emisión del listado son los siguientes:

Ficha RangosFicha Rangos

Cliente

Código de cliente cuyas novedades de facturación asociadas se desean incluir en el informe.

Puede seleccionarse un cliente en particular o un grupo de ellos. Si este dato es dejado en blanco, se incluirán las novedades de todos los clientes en el informe.

Este dato no se encuentra habilitado si se seleccionan novedades de atributos.

Atributo

Código de atributo de clientes cuyas novedades asociadas se desean incluir en el informe.

Si este dato es dejado en blanco, se incluirán las novedades de todos los atributos de clientes en el informe.

Este dato no se encuentra habilitado si se seleccionan novedades sólo de clientes (específicas).

Valor

Valor del atributo de clientes indicado, cuyas novedades asociadas se desean incluir en el informe.

Este dato sólo se encuentra habilitado si se selecciona un atributo de clientes para listar las novedades.

Tipo de novedad

Indicación del tipo de novedades que se desean incluir en el informe.

Cliente (específicas)

Si se selecciona novedades específicas de clientes, se lista para cada cliente el detalle de las novedades que se le ingresaron.

Atributo

Si se selecciona novedades de atributos de clientes, se lista para cada atributo y valor de atributo de clientes el detalle de las novedades ingresadas.

Cliente (totales)

Si se selecciona novedades totales de clientes, se lista para cada cliente el detalle de las novedades que se le ingresaron más las que se ingresaron para cada atributo y valor de atributo válido para el cliente.

Procesamiento

Indicación del tipo de procesamiento de las novedades que se desean incluir en el informe.

El tipo de procesamiento es el que determina en qué tipos de procesos será incluida automáticamente cada novedad.

Puede seleccionarse un tipo en particular o todos los que se deseen.

Manual

Si se selecciona tipo Manual se incluyen las novedades ingresadas para ser incluidas en la facturación manual.

Proceso Nro. XX (XX puede ser un número de 1 a 12)

Si se selecciona tipo Proceso XX se incluyen las novedades ingresadas para ser incluidas en los procesos de facturación automática cuando se indique tipo de proceso XX.

 

EmpresasEmpresas

Permite indicar las empresas a tener en cuenta. En caso de indicar varias, el informe incluirá información de todas ellas.

Descripción

Texto que encabezará los informes multiempresa.

Completar con empresas

Presionando este botón, el cuadro "Descripción" se completa con los nombres de las empresas habilitadas, separados por el caracter "/".

Empresas

Permite informar las empresas a incluir en el informe.

Empresa/Nombre

Se exhiben las empresas habilitadas.

Para cada una de ellas se debe indicar, mediante un tilde, si se desea incluir su información en el informe.

Invertir selección

Presionando el botón se invierte la selección existente de empresas a incluir en el informe, es decir, se deshabilitan las habilitadas, y viceversa.
