Definición de componentes de presupuesto de compras

En esta ventana se definen las componentes de cada uno de los presupuestos de compras que se hayan definido.

Al definir una componente se indica, para determinados valores del atributo asociado al presupuesto, el importe de la compra en la moneda indicada en la definición del presupuesto, si corresponde se indicará la cantidad involucrada en la compra (expresada en la unidad de medida de compras del bien y el período de vigencia del componente (que deberá estar comprendido en el período indicado en la definición del presupuesto).

Es posible parametrizar el sistema para que las componentes de los presupuestos requieran de autorización.

En caso que una componente autorizada se modifique, quedará nuevamente pendiente de autorización.

Una vez que el presupuesto se encuentre aprobado, no se podrán modificar ni eliminar las componentes ingresadas previas a la aprobación de dicho presupuesto; éstas se denominan componentes originales.

Toda modificación al presupuesto que se requiera realizar se deberá registrar en componentes ingresadas con posterioridad a la aprobación del mismo. Estas se consideran como modificaciones al presupuesto original y, si el sistema está parametrizado para que las componentes requieran de autorización, quedarán pendientes de autorización.

Empresa

Empresa correspondiente al presupuesto.

Presupuesto

Código del presupuesto para el cual se define la componente.

Un presupuesto puede tener todas las componentes que se requieran.

Componente

Código identificatorio del componente que se define.

Descripción

Texto que identifica a la componente.

Atributo

Exhibe el atributo que corresponde al presupuesto.

La componente que se está definiendo presupuestará cantidades e importes en relación a los valores de este atributo.

Totales

Este cuadro exhibe los totales resultantes de los importes y, si corresponde, cantidades presupuestados en la componente que se define.

Componentes

Valor

Código del valor del atributo que define al presupuesto para el cual se está presupuestando el importe y la cantidad.

Luego, al registrar compras de bienes y servicios que tengan asociados este valor para el atributo en cuestión, si el grupo de compras de los mismos indica que se debe realizar control de presupuesto de compras, se realizará el correspondiente control.

Descripción

Nombre correspondiente al valor del atributo informado.

Proveedor

Si el presupuesto admite apertura por proveedor, es posible indicar el código del proveedor en relación al cual se está presupuestando esa línea de la componente.

Para cada valor del atributo es posible indicar varias líneas involucrando proveedores diferentes. También se indicará un valor adicional al cual sumen todos los proveedores no contemplados en la apertura.

Nombre

Si se informó un código de proveedor, se exhibe el nombre correspondiente.

Importe

Importe presupuestado (expresado en la moneda indicada en la definición del presupuesto) para el período de vigencia de la componente.

Cantidad

Cantidad de la compra (si corresponde), en la unidad de medida de compras, siempre que se haya indicado Importes y cantidades en la definición del presupuesto.

Fecha Desde

Fecha inicial del período de vigencia de la componente (debe ser posterior o igual a la fecha de inicio del presupuesto).

Fecha Hasta

Fecha final del período de vigencia de la componente (debe ser anterior o igual a la fecha final del presupuesto).
