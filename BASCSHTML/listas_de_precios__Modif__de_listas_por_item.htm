Modificación de listas por ítem

Mediante esta tarea es posible modificar el precio de un mismo bien en una o más listas de precios que lo incluyan, borrarlo de dichas listas o agregarlo a una lista de precios que no lo tenga incluido. En éste último caso el ítem debe cumplir con los valores del atributo definidos para dicha lista de precios.

Sólo pueden modificarse listas normales, es decir, que no se haya indicado Lista dinámica en su definición.

Es importante tener en cuenta que solo pueden modificarse los precios de listas de precios con fechas de vigencia iguales o posteriores a la fecha del día. Ahora bien, si se trata de listas de precios no habilitadas para ventas, si podrán modificarse los precios correspondientes a vigencias anteriores.

En caso de haberse implementado un circuito de autorización de listas de precios, también podrán modificarse los precios de listas habilitadas para ventas con fechas de vigencia anteriores a la fecha del día siempre y cuando dichas listas se encuentren pendientes de autorizar.

Para realizar la tarea se deben completar los siguientes datos:

 

ÍtemÍtem

Código del ítem cuyos precios se desea modificar o eliminar.

Para acceder rápidamente a los precios del ítem, es posible luego de informar su código pulsar el botón Precarga, de esta manera se exhiben en pantalla todas las listas que lo incluyen. Si se indica un código alternativo del ítem, éste se sustituirá por el código de ítem correspondiente.

Descripción largaDescripción larga

Descripción del ítem informado en el campo precedente.

 

Patrón de búsquedaPatrón de búsqueda

Lista

Código de la lista de precios cuyos valores se desea modificar. Es posible establecer filtros en relación a las listas de precios que se han de exhibir en la precarga. Para ello se deberá indicar uno o más códigos de listas de precios en este campo.

Precarga

Presionando este botón se cargarán en pantalla todas las listas de precios vigentes a la fecha, que incluyen el ítem indicado (siempre y cuando se cumplan las condiciones que permiten la modificación sus precios) y las listas de precios con vigencias posteriores a la fecha actual.

Incluir listas de precios con vigencia expirada

Indicación de si se desea incluir entre las listas de precios que se exhiban, aquellas cuya vigencia se encuentra expirada.

La tarea también se exhibe:

Las listas de precios con vigencia anterior a la fecha actual si las mismas no están habilitadas para ventas y no son la lista de precios de valuación contable.

Las listas de precios con vigencia anterior a la fecha actual, si las mismas no están habilitadas para ventas y son la lista de valuación contable pero la vigencia es posterior al último cierre operativo de los subdiarios de materiales.

Tabla de listas de preciosTabla de listas de precios

Para cada lista de precios se exhibe su código, descripción, vigencia, precio asociado y la descripción de la unidad de medida que considera la lista.

Si el ítem contiene precios distintos por color y medida, al hacer clic sobre la columna Precio se abre una ventana con todas las combinaciones de colores y medidas y el precio por cada una de ellas.

En la precarga a partir de una lista de precios se realizarán las conversiones necesarias en el caso que la unidad de medida de las listas sea diferente.

Para eliminar un ítem de la lista bastará con hacer doble clic en la cabecera de la fila de modo que la misma quede marcada para borrarlo.

Para modificar algún precio bastará con digitarlo directamente.

Una vez aprobada la tarea, la lista de precios se habrá actualizado con los precios informados.

Para tener en cuentaPara tener en cuenta

Si el ítem contiene precios distintos por color y medida, al hacer clic sobre la columna Precio, se abre una ventana con todas las combinaciones de colores y medidas y el precio por cada una de ellas.

Si el ítem tiene precios iguales por color pero distintos por medida, se agrega una fila de precios al principio de la tabla, que permite el ingreso de los distintos precios por medida que se copian a todos los colores cuya combinación esté habilitada y se muestran en color gris. No se permite el ingreso de precios en las filas de colores.

Si el ítem tiene precios distintos por color pero iguales por medida, se agrega una columna de precios al principio de la tabla, que permite el ingreso de los distintos precios por color que se copian a todas las medidas cuya combinación esté habilitada y se muestran en color gris. No se permite el ingreso de precios en las columnas de medidas.
