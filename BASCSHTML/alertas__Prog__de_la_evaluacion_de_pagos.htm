Evaluación de pagos 

Esta tarea permite programar la evaluación de pagos en forma automática. Esto genera información de pagos para la agenda de los usuarios parametrizados para recibir dicha información.

Las opciones para programar este proceso son:

Periódico

A través de esta selección es posible programar la tarea para que se realice en base a un período determinado (días, semanas, meses).

Sucede

Diariamente

Semanalmente

Mensualmente

En el primer caso es posible parametrizar el sistema para que el proceso se ejecute cada n cantidad de días.

Si se selecciona la opción semanalmente se puede parametrizar para que el proceso se ejecute cada  n semanas o en determinados días de la semana.

Si se selecciona la opción mensualmente se parametriza para que el proceso se ejecute cada n meses o determinado día de cada mes, o el primer (segundo, tercero, cuarto o último) día, por ejemplo martes, de cada mes.

Frecuencia diaria

En base a esta selección se programa el proceso para que se efectúe diariamente. Las posibilidades son:

Sucede una vez a las:

Sucede cada:

n horas o minutos, comenzando a las hora determinada finalizando a las hora determinada. (Por ejemplo: de 8:00:00 a 18:00:00 hs.).

Una vez

Si se indica esta opción el proceso se ejecuta un sola vez el día y hora indicadas.

Agregar salida al historial de la tarea

Si se tilda esta opción se emite un graban los pasos del proceso que se ejecuta en el SQL.
