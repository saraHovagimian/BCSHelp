Impresión de pedidos

Permite imprimir los comprobantes registrados en el sistema.

La cantidad de copias, formato de impresión, tipo de emisión (impresora o archivo) se definen en el talonario correspondiente.

Independientemente de lo que se indique en el talonario, se dispone de botones para emitir el comprobante por impresora y en un archivo PDF.

El formato de impresión está contenido en un archivo de extensión QRP que contiene los datos del comprobante que se deseen imprimir, la ubicación de los mismos en el formulario, la forma que tiene el encabezado, cuerpo y pie del formulario, etc.

Si se imprime en archivo es posible utilizar una plantilla como fondo. Puede utilizarse un archivo PDF con el preimpreso del formulario de pedido.

Los datos a ingresar para la emisión son:

Desde número Desde número

Número de comprobante del pedido desde el que se desea imprimir.

Deben completarse el prefijo y el número de comprobante. Si el número no se completa, se realizará la impresión desde el primer pedido existente para el prefijo indicado.

El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección.

El prefijo trae asociado el número de talonario de los pedidos a imprimir.

Hasta númeroHasta número

Número de comprobante del pedido hasta el que se desea imprimir.

Deben completarse el prefijo y el número de comprobante. Si el número no se completa, se realizará la impresión hasta el último pedido existente para el prefijo indicado.

El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección.

Si se indica desde y hasta un mismo número de pedido, se realizará la emisión de ese pedido en particular.

Sólo los comprobantes no impresosSólo los comprobantes no impresos

Indicación de si se desean emitir solamente los pedidos que aún no han sido emitidos o todos los que se incluyen en el rango de números indicado.

 

Período a considerar

Para tener en cuentaPara tener en cuenta

El sistema al momento de imprimir verifica si para el rango de números indicado hay comprobantes repetidos en más de una vigencia, avisando al usuario de tal eventualidad sugiriendo indicar un rango de fechas para evitar las repeticiones. Además, se consulta al usuario si desea seguir con la impresión o no.

En caso de continuar con la impresión el sistema imprimirá todos los comprobantes en el rango indicado teniendo en cuenta el rango de fechas aunque haya números de comprobante repetidos, por lo que un mismo número de comprobante podría imprimirse más de una vez para diferentes vigencias de fechas.

En la vista preliminar y emisiones personalizadas de pedidos de ventas y otras entregas, cuyo talonario indica detalle de ubicaciones, que no tengan indicado detallar medidas y colores en forma tabular:

Cuando el depósito administre múltiples ubicaciones se podrán detallar hasta 5 ubicaciones, su descripción y stock actual en unidad de ventas

 Cuando el depósito no administre múltiples ubicaciones sólo se podrá detallar una sola ubicación, su descripción y stock actual en unidad de ventas.

Cabe aclarar, que el depósito que se considera para el detalle de las ubicaciones de ítems ubicados es el depósito asociado a la línea de detalle.
