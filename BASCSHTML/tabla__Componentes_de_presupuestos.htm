Componentes de presupuesto

En esta ventana se definen las componentes de cada uno de los presupuestos que se hayan definido.

Al definir una componente se indica, para determinadas imputaciones contables asociadas al presupuesto, el importe de la compra en la moneda indicada en la definición del presupuesto y el período de vigencia del componente (que deberá estar comprendido en el período indicado en la definición del presupuesto).

Es posible parametrizar el sistema para que las componentes de los presupuestos requieran de autorización.

En caso que una componente autorizada se modifique, quedará nuevamente pendiente de autorización.

Una vez que el presupuesto se encuentre aprobado, no se podrán modificar ni eliminar las componentes ingresadas previas a la aprobación de dicho presupuesto; éstas se denominan componentes originales.

Toda modificación al presupuesto que se requiera realizar se deberá registrar en componentes ingresadas con posterioridad a la aprobación del mismo. Estas se consideran como modificaciones al presupuesto original y, si el sistema está parametrizado para que las componentes requieran de autorización, quedarán pendientes de autorización.
