Emisión de informes desde la Base Informes o desde la Base operativa

Si se ha configurado la Base para informes es posible indicar, para cada usuario, si emitirá los informes de la base operativa, de la base para informes o si se le permitirá seleccionar, en cada caso, de cual de las dos bases desea realizar la emisión.

Si se le permite seleccionar de qué base puede emitir los informes, en cada uno de ellos se exhibe “Cambiar base para informes”. Si se hace clic sobre el texto estando en la base operativa se pasa a la base para informes y viceversa.

En caso que se esté realizando la emisión desde la base para informes se inidca ta situación mediante el mensaje:

“Datos de la base para informes. Ultimo movimiento registrado el dd/mm/aa hh:mm.”
