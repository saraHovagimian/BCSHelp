Precios y bonificaciones digitadas

Este listado informa las modificaciones realizadas a los precios y bonificaciones durante la carga de pedidos y comprobantes de ventas.

Emisión

FCC (Fecha de certeza contable)

Los datos solicitados para la emisión del listado son los siguientes:

Ficha RangosFicha Rangos

Usuario

Nombre del usuario responsable de las modificaciones a listar.

Si este dato es dejado en blanco, se incluyen en el listado las novedades de todos los usuarios.

Fechas

Indicación del tipo de fecha a considerar para la emisión del informe. Las opciones son:

Comprobante

Si se selecciona fecha del Comprobante, se incluirán sólo aquellos comprobantes cuya fecha interna se encuentre comprendida dentro del rango que se indique.

Registración

Si se selecciona fecha de Registración, se incluirán sólo aquellos comprobantes cuya fecha de ingreso al sistema se encuentre comprendida dentro del rango.

Período a considerar

Período

Hora (desde)

Este campo sólo se activa si se seleccionó la opción fecha de Registración.

Aquí se informará la hora a partir de la cual se desean incluir los comprobantes correspondientes a la fecha de registración indicada en el campo anterior. El sistema ofrecerá por defecto la hora 00:00:00.

Hora (hasta)

Este campo sólo se activa si se seleccionó la opción fecha de Registración.

Hora hasta la cual se desean incluir los comprobantes correspondientes a la fecha de registración indicada en el campo anterior. El sistema ofrecerá por defecto la hora 23:59:59.

Si no se indica ninguna hora, se incluyen los comprobantes ingresados hasta el final.

Ficha OpcionesFicha Opciones

Circuito

Indicación de los circuitos o tipos de comprobantes cuyas modificaciones se desea listar.

Pedidos

Para incluir los pedidos en los cuales se modificaron precios o bonificaciones.

Comprobantes de venta

Para incluir los comprobantes que sufrieron modificaciones de precios o bonificaciones..

Tipo

Permite indicar qué modificaciones se desea incluir.

Precios

Si se tilda, sólo se tienen en cuenta los comprobantes que sufrieron modificaciones de precios.

Bonificaciones

Si se tilda, sólo se tienen en cuenta los comprobantes que sufrieron modificaciones de bonificaciones.

Ordenamiento

Tipo de ordenamiento con el que se desean listar los comprobantes en el informe.

Fecha de registración

Los datos se ordenan por la fecha y hora de ingreso al sistema de los comprobantes involucrados.

Para cada fecha, la información se ordena por tipo de comprobante, prefijo y número de comprobante.

Fecha de comprobante

Para cada fecha, la información se ordena por tipo de comprobante, prefijo y número de comprobante.

Talonario

Los datos se ordenan por el talonario al cual corresponden; para cada talonario, por prefijo y número de comprobante. Podrán indicarse además qué talonarios se desea incluir.

Usuario

La información se ordena por operador.

Ítem

Los datos se ordenan por código de ítem, código de color y medida.
