Notas de crédito

Según las necesidades de la empresa, es posible establecer circuitos de trabajo que requieran la autorización de las notas de crédito.

Se puede implementar el sistema de modo que las notas de crédito requieran de autorización para que sea posible darles curso. El que la requieran o no y, en el caso que la requieran, las condiciones que hacen que una nota de crédito deba autorizarse, quedan a elección del usuario.

Es posible indicar una tabla de autorizaciones para todas las notas de crédito en general, e indicar otras tablas, por ejemplo para las notas de crédito de determinados clientes o que incluyen determinados ítems.

También es posible indicar que se deben autorizar las notas de crédito por devolución de determinados bienes o que tienen como titulares a determinados clientes, es posible indicar un atributo y luego informar diferentes tablas para algunos de los valores del mismo. Si no se informan todos los valores del atributo, los comprobantes correspondientes a los valores que no se indiquen se autorizarán automáticamente, es decir, no requerirán de autorización.

Los parámetros pueden modificarse mientras se está operando con el sistema y la actualización de las opciones que se indiquen se realizará en forma inmediata sin necesidad de salir del sistema y volver a entrar.

Los parámetros posibles de configurar son:

Evaluar en función del: comprobante

Permite indicar qué importe se tiene en cuenta para autorizar, puede seleccionarse:

Total

Total gravado

Para todas las notas de crédito

Indicación de si se desea que todas las notas de crédito se autoricen. La forma en que deber efectuarse la autorización dependerá de la tabla de autorizaciones que se indique.

Tabla de autorizaciones

Tabla que se utiliza para autorizar todas las notas de crédito.

Con ítems determinados

Indicación de si las notas de crédito por devolución de ítems determinados requieren o no autorización manual.

Atributo

Código de atributo. Debe corresponder a un atributo ya definido en el sistema y aplicable a Ítems.

Esta opción sólo se habilita si se indicó la autorización manual de notas de crédito.

Las notas de crédito que incluyan ítems que tengan este atributo serán tenidos en cuenta para decidir si el comprobante requiere o no autorización. El que la requieran o no y la forma en que deberá efectuarse la misma dependerá de los valores del atributo que se informen en la tabla que sigue y de la tabla de autorizaciones que se indique para cada uno de dichos valores.

Tabla de valores

En esta tabla se deberán ingresar los distintos valores del atributo de ítems, indicado en el campo precedente, para los cuales se desea definir un circuito de autorización.

Valor

Valor del atributo cuyo código se indicó en el campo Atributo.

Esta opción sólo se habilita si se indicó la autorización manual de notas de crédito Con ítems determinados.

Las notas de crédito que incluyan ítems que tengan este valor para el atributo indicado en el campo correspondiente, requerirán autorización y la forma en que debe efectuarse la misma dependerá de la tabla de autorizaciones que se indique.

Con textos varios determinados

Permite indicar criterios de autorización por textos varios del tipo importe o porcentaje. Requerirán autorización manual los comprobantes  que se ingresen con algún ítem texto vario de tipo importe o porcentaje y que cumpla además con los demás criterios indicados.

De clientes determinados

Indicación de si las notas de crédito que tengan como titulares a clientes determinados requieren o no autorización manual.

Atributo

Código de atributo. Debe corresponder a un atributo ya definido en el sistema y aplicable a Clientes.

Esta opción sólo se habilita si se indicó que la autorización manual de notas de crédito De Clientes determinados.

Las notas de crédito que tengan este atributo serán tenidos en cuenta para decidir si el comprobante requiere o no autorización. El que la requieran o no y la forma en que deberá efectuarse la misma dependerá de los valores del atributo que se informen en la tabla que sigue y de la tabla de autorizaciones que se indique para cada uno de dichos valores.

Tabla de valores

En esta tabla se deberán ingresar los distintos valores del atributo de clientes, indicado en el campo precedente, para los cuales se desea definir un circuito de autorización.

Valor

Valor del atributo cuyo código se indicó en el campo anterior.

Esta opción sólo se habilita si se indicó que la autorización manual de notas de crédito se requiere para Clientes determinados.

Las notas de crédito que tengan este valor para el atributo indicado en el campo anterior, requerirán autorización y la forma en que debe efectuarse la misma dependerá de la tabla de autorizaciones que se indique.

Destinatarios

Es posible indicar a quien/es se les envía email cuando una nota de crédito queda totalmente autorizada o rechazada.

Al autorizar

Una o mas direcciones de email a las que se envía aviso al autorizar totalmente una nota de crédito. Puede indicarse una dirección de email o un tag que permita obtener una dirección de email. Por ejemplo,el tag "Vendedor" indica que se utilice la dirección de email del vendedor del comprobante.

Al rechazar

Una o mas direcciones de email a las que se envía aviso al rechazar totalmente  una nota de crédito. Puede indicarse una dirección de email o un tag que permita obtener una dirección de email. Por ejemplo,el tag "Vendedor" indica que se utilice la dirección de email del vendedor del comprobante.
