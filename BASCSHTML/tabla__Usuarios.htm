Usuarios

El sistema permite modificar el formato y el contenido de los menúes y submenúes del sistema, según las necesidades de cada empresa. Se pueden habilitar o deshabilitar tareas, cambiar las ubicaciones de las tareas en el menú o modificar los títulos con los que aparecen las opciones en el menú.

Otra prestación que brinda el sistema es la de habilitar para cada usuario cajas , prefijos de tipos de comprobantes o talonarios, depósitos y conceptos específicos que son los únicos que podrán utilizarse por el mismo. Esta prestación es opcional y permite acotar el rango de cajas, talonarios, depósitos o conceptos que pueden ser utilizados por cada usuario del sistema. Si se trabaja con múltiples empresas, también es posible habilitar para cada usuario, las empresas y sucursales con las que podrá operar.

Tanto la configuración de los menúes como los rangos de cajas, prefijos, depósitos, conceptos y empresas pueden especificarse para un usuario determinado o para un grupo de usuarios. Si un usuario no tiene dada una configuración específica, usará la del grupo de usuarios al que pertenece.

Como ya mencionamos, el sistema brinda la posibilidad de definir distintos menúes para los distintos operadores que trabajan con el sistema. Esto es muy útil si se tiene en cuenta que hay distintos operadores para ejecutar los diversos procesos del sistema, y de esta forma, cada uno puede tener habilitadas las tareas que necesita realizar. Por ejemplo, tareas vinculadas con las compras, facturación, cobranzas, pagos a proveedores, etc.

Al empezar a utilizar el sistema todas las tareas del mismo se encuentran disponibles para todos los usuarios del sistema y sólo a través de este punto podrán definirse las distintas configuraciones de menúes a utilizar.

El sistema se entrega con un menú standard definido para el usuario sa del grupo de usuarios public. Los distintos usuarios del sistema utilizarán este menú mientras no se les configure un menú distinto. Para los usuarios que tengan configurado un menú especial, cada vez que se efectúe una modificación en dicho menú, la misma afectará sólo al menú del usuario especificado y no al resto. Si un grupo de usuarios tiene configurado un menú especial, cada vez que se efectúe una modificación en dicho menú, la misma afectará a todos los usuarios del grupo salvo a aquellos que a su vez tengan un menú especial definido.

Se permite además cambiar la ubicación de una tarea pasándola de un submenú a otro en el que resulte más práctico y cómodo según el tipo de operatoria de la empresa, además es posible cambiar el título de un menú, de un submenú, o de una tarea del sistema.

La pantalla tiene una serie de botones que facilitan y permiten agilizar el proceso de configuración de menúes.

Grupos/usuarios Grupos/usuarios

Se muestra una lista conteniendo todos los usuarios habilitados para cada grupo de usuarios definido.

Se permite seleccionar de esta lista, el usuario o el grupo de usuarios para el cual se desea configurar o modificar el menú del sistema.

También puede seleccionarse un usuario para habilitarle cajas, depósitos, talonarios, conceptos o empresas definidas en el sistema

Ficha MenúsFicha Menús

Arriba

Mediante este botón se sube un lugar el punto del menú o el submenú en el que está posicionado el cursor.

Cada vez que se pulsa el botón la tarea o el submenú se desplaza un lugar hacia arriba, pero siempre dentro del mismo nivel en el que se está posicionado.

Por ejemplo, si erróneamente se ubicó una tarea en un lugar que no es el deseado, en lugar de borrar la tarea de este lugar e ingresarla nuevamente en el lugar correcto, es posible utilizar este botón para desplazar la misma hasta el lugar deseado.

Abajo

Mediante este botón se baja un lugar el punto del menú o el submenú en el que está posicionado el cursor.

Cada vez que se pulsa el botón la tarea o el submenú se desplaza un lugar hacia abajo, pero siempre dentro del mismo nivel en el que se está posicionado.

Por ejemplo, si erróneamente se ubicó una tarea en un lugar que no es el deseado, en lugar de borrar la tarea de este lugar e ingresarla nuevamente en el lugar correcto, es posible utilizar este botón para desplazar la misma hasta el lugar deseado.

Izquierda

Mediante este botón se desplaza un lugar hacia la izquierda el punto del menú o el submenú en el que está posicionado el cursor.

Cada vez que se pulsa el botón la tarea o el submenú se desplaza un lugar hacia la izquierda, es decir que se ubica en el nivel inmediato superior.

Por ejemplo, si erróneamente se ubicó una tarea bajo un submenú que no es el correcto, en lugar de borrar la tarea de este lugar e ingresarla nuevamente en el lugar correcto, es posible utilizar este botón para pasar la misma al nivel de arriba.

Derecha

Mediante este botón se desplaza un lugar hacia la derecha el punto del menú o el submenú en el que está posicionado el cursor.

Cada vez que se pulsa el botón la tarea o el submenú se desplaza un lugar hacia la derecha, es decir que se ubica en el nivel inmediato inferior.

Por ejemplo, si erróneamente se ubicó una tarea bajo un submenú que no es el correcto, en lugar de borrar la tarea de este lugar e ingresarla nuevamente en el lugar correcto, es posible utilizar este botón para pasar la misma al nivel de abajo.

Expandir

Mediante este botón se abre en pantalla todo el detalle del menú que se está configurando.

Se observan todos los submenúes y tareas de cada nivel con el nombre que cada una tiene definido en el menú del sistema.

Colapsar

Mediante este botón se cierra el detalle del menú y sólo quedan en pantalla los títulos de los submenúes del mismo.

Agregar Submenú

Mediante este botón es posible agregar un nuevo submenú en el menú que se está configurando.

Se abre una ventana que posibilita el ingreso del título del submenú y una vez ingresado el mismo, se debe pulsar la tecla F10 para confirmar el ingreso del nuevo submenú.

El submenú se agrega dentro del menú o submenú desde el cual se pulsó el botón, y mediante los botones de desplazamiento es posible cambiar la ubicación del mismo.

Agregar Programa

Mediante este botón es posible agregar una tarea en el menú que se está configurando.

Se abre una ventana que posibilita la selección de la tarea de la lista de programas posibles y el ingreso del título que tendrá la misma en el menú. Una vez ingresado el mismo, se debe pulsar la tecla F10 para confirmar el ingreso del nuevo programa.

El programa se agrega dentro del menú o submenú desde el cual se pulsó el botón, y mediante los botones de desplazamiento es posible cambiar la ubicación del mismo.

Programa externo

Mediante este botón es posible agregar una tarea que se ejecuta en el sistema invocando a un programa externo al mismo.

Se abre una ventana que posibilita el ingreso de la línea de comandos que debe ejecutarse al seleccionar la tarea del menú y se debe ingresar el título que tendrá la misma en el menú. Una vez ingresados estos datos, se debe pulsar la tecla F10 para confirmar el ingreso del nuevo programa externo.

El programa se agrega dentro del menú o submenú desde el cual se pulsó el botón, y mediante los botones de desplazamiento es posible cambiar la ubicación del mismo.

Separador

Mediante este botón es posible agregar una línea de separación entre las tareas definidas para un determinado menú o submenú del sistema.

Esta separación es útil en menúes o submenúes muy extensos para agrupar por algún criterio las tareas del mismo y visualizar el menú de una forma mas ordenada.

Modificar

Mediante este botón es posible modificar los programas, los títulos de los mismos o los títulos de los submenúes definidos en el menú.

Al pulsar el botón estando el cursor posicionado en un submenú, se abre una ventana que exhibe la descripción correspondiente al mismo y se permite su modificación.

Si se trata de un programa la ventana muestra el nombre del mismo y el título que tiene definido en el menú y es posible modificar ambos valores. El título que se indique será el que aparecerá como opción del menú que se está definiendo.

Borrar

Mediante este botón se elimina del menú el submenú o el programa en el que está posicionado el cursor y el mismo no podrá ser utilizado por el usuario o grupo para el cual se elimina.

Al eliminar un submenú se eliminan también todos los programas que lo integran.

Ficha PrefijosFicha Prefijos

Mediante esta ficha es posible habilitar prefijos para los distintos tipos de comprobantes posibles de utilizar. El prefijo determina el talonario que será utilizado por el tipo de comprobante.

Pueden habilitarse prefijos de comprobantes para los distintos usuarios o grupos de usuarios definidos y para las distintas empresas en las cuales el usuario puede operar.

Si un usuario no tiene indicado nada en esta ficha podrá utilizar todos los prefijos y talonarios habilitados para el grupo de usuarios al que pertenece, y si el grupo no tiene nada indicado en esta ficha, el usuario podrá utilizar todos los prefijos y talonarios definidos en el sistema para cada tipo de comprobante.

Si se habilitan tipos de comprobante, el usuario sólo podrá ingresar dichos comprobantes con el prefijo que se le indique. Es decir, cuando el usuario ingrese comprobantes de los tipos indicados, se validará que se utilicen para los mismos sólo los prefijos indicados.

Se muestran dos tablas, una para indicar el tipo de habilitación para la operatoria de ingreso de comprobantes, y la otra para la eliminación o anulación de los mismos.

Para el caso de ingresos de comprobantes se puede definir si el usuario o grupo de usuarios tienen todos los talonarios habilitados marcando la opción Todos habilitados. Si se desea se puede indicar talonarios exceptuados a esta indicación, es decir, aquellos que no se encuentran habilitados.

Si el usuario o grupo de usuarios no tuviera ningún talonario habilitado para ingresar comprobantes se marca la opción Ninguno habilitado. Si se desea se puede definir talonarios exceptuados a este requisito, es decir, aquellos que están permitidos o habilitados para la carga de comprobantes.

Mismo mecanismo se aplica para habilitar eliminar o anular comprobantes según el talonario.

Si el usuario o grupo de usuarios tienen todos los talonarios habilitados entonces se marca la opción Todos habilitados. Si se desea exceptuar algún talonario se indica en la tabla Excepto los indicados.

Si el usuario o grupo de usuarios no tuviera ningún talonario habilitado para eliminar o anular comprobantes se marca la opción Ninguno habilitado. Si se desea se puede definir aquellos talonarios exceptuados, es decir, aquellos que están permitidos o habilitados para la eliminación y/o anulación.

Ficha Datos generalesFicha Datos generales

En esta ficha es posible indicar información general del usuario o grupo de usuarios seleccionado.

Nombre

Nombre del usuario o grupo de usuarios seleccionado.

Puede ingresarse un texto de hasta treinta caracteres.

E-mail

E-mail del usuario o grupo de usuarios seleccionado.

Se dispone de un campo de hasta cincuenta caracteres para el ingreso.

Empresa default

Empresa a la que ingresa el usuario al entrar al sistema.

Si no se trabaja con múltiples empresas la empresa default será siempre la 1.

Es un número de hasta dos dígitos que debe corresponder a una empresa definida en el sistema y el sistema verificará que el usuario esté habilitado para operar en la misma.

Todas las transacciones y procesos efectuados por el usuario se registrarán en la empresa que aquí se indique. Esta empresa puede ser modificada por otra cada vez que se necesite, ya sea desde este punto o desde el punto Parámetros de usuarios, y sólo afecta a las nuevas transacciones que registre el usuario, y no a las registradas por el mismo en otras empresas.

Sucursal default

Sucursal de la empresa default a la que ingresa el usuario al entrar al sistema.

Si no se trabaja con múltiples empresas la sucursal default será siempre la 1.

Es un número de hasta dos dígitos que debe corresponder a una sucursal definida en el sistema para la empresa indicada en el dato anterior y se verificará que el usuario esté habilitado para operar en la misma.

Todas las transacciones y procesos efectuados por el usuario se registrarán en la sucursal de la empresa default que aquí se indique. Esta sucursal puede ser modificada por otra cada vez que se necesite, ya sea desde este punto o desde el punto Parámetros de usuarios, y sólo afecta a las nuevas transacciones que registre el usuario, y no a las registradas por el mismo en otras empresas y sucursales.

También podrá dejarse este campo en blanco si se quiere que el usuario puede ingresar comprobantes para las distintas sucursales de su empresa en forma simultánea, sin tener que cambiar de sucursal cada vez.

Agenda

También se indica la cantidad de días de anticipación con que se desea que se muestren las tareas pendientes en la Agenda.

Tiempo de refrescado  n minutos

En este campo se informa en minutos el tiempo de refrescado de las agendas de cada uno de los usuarios que se hayan definido en el campo Grupos/Usuarios.

Mostrar tareas con n días de anticipación

En este campo es posible indicar la cantidad de días de anticipación con que se desea que se muestren las tareas pendientes en la Agenda. Ejemplo: 1 día de anticipación.

Si no se indica nada en este campo, en la agenda se muestran todas las tareas pendientes del usuario sin tener en cuenta la fecha de vencimiento.

Ficha EmpresasFicha Empresas

Esta ficha se utiliza si el sistema se opera con múltiples empresas.

Aquí es posible habilitar empresas y sucursales para ser utilizadas por el usuario o grupo de usuarios seleccionado.

Por default un usuario podrá utilizar todas las empresas y sucursales habilitadas para el grupo de usuarios al que pertenece, y el grupo por default podrá utilizar todas las empresas y sucursales ingresadas en el sistema.

Un usuario podrá tener empresas habilitadas para ingresar comprobantes, para emitir informes o para ambas tareas.

Si se habilitan empresas y sucursales en esta ficha, el usuario sólo podrá registrar comprobantes en las empresas y sucursales que se indiquen y sólo podrá emitir informes en los que participen las empresas y sucursales que se habiliten para informes.

Tabla de empresas

Esta tabla muestra la lista de empresas posibles de seleccionar para ser utilizadas por el usuario. Por default en esta lista se ofrecen marcadas como habilitadas (en color rojo) todas las empresas, de manera de no tener que acceder a esta ficha si se habilita al usuario para operar en todas las empresas.

Las empresas no se ingresan manualmente en la tabla, sino que la misma se completa automáticamente con las empresas definidas en el sistema, y podrá habilitarse o deshabilitarse cada empresa haciendo doble click en la fila de la tabla correspondiente a la misma.

Invertir Selección

Pulsando el botón Invertir selección las empresas habilitadas se deshabilitan y las no habilitadas se marcan automáticamente como habilitadas.

Código

Código de la empresa que se desea habilitar para ser utilizada por el usuario seleccionado.

Por ejemplo, se puede habilitar una empresa para registrar comprobantes y para emitir informes, pero habilitar para registrar comprobantes a una sola sucursal de la empresa. De esta forma, aunque la empresa se encuentre habilitada para registrar comprobantes, sólo podrán registrarse los mismos en la sucursal habilitada.

Descripción

Descripción correspondiente a la empresa indicada.

Habilitada para comprobantes

Indicación de si la empresa indicada se encuentra habilitada para registrar comprobantes, es decir, si el usuario seleccionado podrá o no ingresar comprobantes en la empresa.

Al habilitar una empresa para registrar comprobantes, se habilitan automáticamente todas las sucursales de la misma, pudiendo luego deshabilitar las sucursales de la empresa que se deseen.

Al deshabilitar una empresa para registrar comprobantes, se deshabilitan automáticamente todas las sucursales de la misma, y no podrá ser posible habilitar para comprobantes a ninguna sucursal de la empresa.

Habilitada para informes

Indicación de si la empresa indicada se encuentra habilitada para emitir informes, es decir, si el usuario seleccionado podrá o no emitir información de la empresa.

Al habilitar una empresa para emitir informes, se habilitan automáticamente todas las sucursales de la misma, pudiendo luego deshabilitar las sucursales de la empresa que se deseen.

Al deshabilitar una empresa para emitir informes, se deshabilitan automáticamente todas las sucursales de la misma, y no podrá ser posible habilitar para informes a ninguna sucursal de la empresa.

Tabla de sucursales

Esta tabla muestra la lista de sucursales posibles de seleccionar para ser utilizadas por el usuario en cada empresa.

Las sucursales no se ingresan manualmente en la tabla, sino que la misma se completa automáticamente con las sucursales definidas en el sistema, y podrá habilitarse o deshabilitarse cada sucursal de una empresa haciendo doble click en la fila de la tabla correspondiente a la misma.

Pulsando el botón Invertir selección las sucursales habilitadas se deshabilitan y las no habilitadas se marcan automáticamente como habilitadas.

Código

Código de la sucursal que se desea habilitar para ser utilizada por el usuario seleccionado.

Este dato no se ingresa manualmente, sino que se completa automáticamente al posicionarse en una empresa en la tabla de empresas explicada anteriormente. Al posicionar el cursor en una empresa en dicha tabla, se completa automáticamente la tabla de sucursales con la lista de sucursales de la empresa, permitiendo de esta forma, habilitar o deshabilitar sucursales de las empresas.

Por ejemplo, se puede habilitar una empresa para emitir informes, pero sólo de un grupo de sucursales de la misma. Para esto una vez habilitada la empresa se pueden deshabilitar las sucursales que se deseen. De esta forma, aunque la empresa se encuentre habilitada para informes, sólo se incluirá en los mismos la información de las sucursales que se hayan habilitado.

Descripción

Descripción correspondiente a la sucursal cuyo código se exhibe en el dato anterior.

Habilitada para comprobantes

Indicación de si la sucursal indicada se encuentra habilitada para registrar comprobantes, es decir, si el usuario seleccionado podrá o no ingresar comprobantes en la sucursal.

Habilitada para informes

Indicación de si la sucursal indicada se encuentra habilitada para emitir informes, es decir, si el usuario seleccionado podrá o no emitir información de la sucursal.
