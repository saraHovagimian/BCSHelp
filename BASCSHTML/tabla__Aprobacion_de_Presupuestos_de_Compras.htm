Aprobación

La tarea de aprobación de un presupuesto es necesaria y lo habilita para que puedan imputársele compras.

Para poder imputar compras a un presupuesto es necesario aprobarlo, mediante la tarea Aprobación de presupuestos de compras.

Para poder aprobar un presupuesto se requiere que todas las componentes del mismo se encuentren autorizadas. Vale aclarar también que la tarea de aprobación requiere que el presupuesto tenga definida, al menos, una componente.

Al aprobar un presupuesto éste pasa del estado En preparación al estado Aprobado.

PresupuestoPresupuesto

Para realizar la tarea se debe informar, en el campo Presupuesto, el presupuesto a cerrar.

Al informar el código, el sistema exhibe en la pantalla toda la información asociada a la definición del presupuesto y presionando F10-Grabar, el presupuesto queda aprobado.

Una vez que el presupuesto se encuentre aprobado, no se podrán modificar ni eliminar las componentes ingresadas previas a la aprobación de dicho presupuesto; éstas se denominan componentes originales.

Toda modificación al presupuesto que se requiera realizar se deberá registrar en componentes ingresadas con posterioridad a la aprobación del mismo. Estas se consideran como modificaciones al presupuesto original y, si el sistema está parametrizado para que las componentes requieran de autorización, quedarán pendientes de autorización.

Una vez aprobado un presupuesto, es posible imputar compras al mismo.
