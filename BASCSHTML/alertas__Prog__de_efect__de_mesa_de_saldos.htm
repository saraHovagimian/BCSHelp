Efectivización mesa de saldos

Este programa permite definir la programación del proceso que efectivizará las mesas de saldos pendientes de depósitos correspondientes a casa central.

Para programar la ejecución de este proceso es necesario completar los datos que se solicitan en esta pantalla, tales como la periodicidad del mismo y la frecuencia diaria, se podrá configurar para que se ejecute siempre un día y a una determinada hora o se pueden indicar períodos de días, semanas o meses pasados los cuales se ejecutará el proceso.

La modificación de la programación periódica del proceso de evaluación sólo podrá ser realizada por el administrador del sistema y podrá modificarse esta programación todas las veces que sea necesario.

En el primer caso es posible parametrizar el sistema para que el proceso se ejecute cada n cantidad de días.

Si se selecciona la opción semanalmente se puede parametrizar para que el proceso se ejecute cada n semanas o en determinados días de la semana.

Si se selecciona la opción mensualmente se parametriza para que el proceso se ejecute cada n meses o determinado día de cada mes, o el primer (segundo, tercero, cuarto o último) día, por ejemplo martes, de cada mes.
