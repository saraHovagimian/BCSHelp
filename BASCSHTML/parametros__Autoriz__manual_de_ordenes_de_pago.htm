Autorización de Ordenes de Pago 

El sistema le brinda la posibilidad al usuario de definir múltiples criterios para la autorización de órdenes de pago.

Se puede implementar el sistema de modo que las ordenes de pago requieran de autorización para que sea posible darles curso. El que la requieran o no y, en el caso que la requieran, las condiciones que hacen que una orden de pago deba autorizarse, quedan a elección del usuario.

Es posible indicar una tabla de autorizaciones para todas las ordenes de pago en general, y/o indicar otras tablas, por ejemplo para las ordenes de pago destinadas a determinados proveedores o a determinados agentes, o a determinados clientes.

Los parámetros pueden modificarse mientras se está operando con el sistema y la actualización de las opciones que se indiquen se realizará en forma inmediata sin necesidad de salir del sistema y volver a entrar.
