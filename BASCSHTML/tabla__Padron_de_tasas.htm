Padrón de tasas

Esta tabla permite definir los padrones a importar destinados al cálculo directo de percepciones y retenciones de Ingresos brutos.

La grilla inferior, Provincias, exhibe las empresas-provincias a las que se ha asociado el padrón; esta información no puede ingresarse directamente desde acá.

Debe tenerse en cuenta que no se puede eliminar un padrón si hay provincias que lo utilizan para el cálculo directo.

CódigoCódigo

Para cada padrón a importar se debe indicar el código y la descripción que le correspondan. El código de padrón se deberá informar en la importación de los datos para ingresos brutos.

Este código se deberá informar, en la tabla de provincias, para aquellas provincias que realizan percepciones y/o retenciones por cálculo directo.
