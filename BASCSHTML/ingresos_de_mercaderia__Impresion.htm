Impresión

Permite imprimir los comprobantes registrados en el sistema.

La cantidad de copias, formato de impresión, tipo de emisión (impresora o archivo) se definen en el talonario correspondiente.

Independientemente de lo que se indique en el talonario, se dispone de botones para emitir el comprobante por impresora y en un archivo PDF.

El formato de impresión está contenido en un archivo de extensión QRP que contiene los datos del comprobante que se deseen imprimir, la ubicación de los mismos en el formulario, la forma que tiene el encabezado, cuerpo y pie del formulario, etc.

Si se imprime en archivo, si se desea, se podrá utilizar una plantilla a utilizar como fondo.

Si el comprobante incluye ítems que se administran mediante números de serie, éstos pueden mostrarse separados por un conjunto de caracteres y/o exhibirse de a uno por línea. Para ello habrá que hacer una parametrización acorde en los Parámetros de Datos Generales y diseñar un formato personalizado de emisión que incluya las variables que correspondan.

Los datos a ingresar para la emisión son:

 

ProveedorProveedor

Si se desea imprimir los comprobantes que tienen por titular a un proveedor determinado, se debe informar su código en este campo.

 

Desde númeroDesde número

Número de comprobante del ingreso de mercadería desde el que se desea imprimir.

Deben completarse el prefijo y el número de comprobante. Si el número no se completa, se realizará la impresión desde el primer ingreso de mercadería existente para el prefijo indicado.

El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección.

Hasta númeroHasta número

Número de comprobante del ingreso de mercadería hasta el que se desea imprimir.

Deben completarse el prefijo y el número de comprobante. Si el número no se completa, se realizará la impresión hasta el último ingreso de mercadería existente para el prefijo indicado.

El prefijo y el número pueden ser completados manualmente o seleccionarse de la lista de comprobantes ingresados que provee el sistema en este punto para permitir la selección.

Sólo los comprobantes no impresosSólo los comprobantes no impresos

Indicación de si se desean emitir solamente los ingresos de mercadería que aún no han sido emitidos o todos los que se incluyen en el rango de números indicado.

 

Período a considerar

Para tener en cuentaPara tener en cuenta

El ingreso de rango de fechas no es obligatorio.

El sistema al momento de imprimir verifica si para el rango de números indicado hay comprobantes repetidos en más de una vigencia, avisando al usuario de tal eventualidad sugiriendo indicar un rango de fechas para evitar las repeticiones. Además, se consulta al usuario si desea seguir con la impresión o no.

En caso de continuar con la impresión el sistema imprimirá todos los comprobantes en el rango indicado teniendo en cuenta el rango de fechas indicado aunque haya números de comprobante repetidos, por lo que un mismo número de comprobante podría imprimirse más de una vez para diferentes vigencias de fechas.
