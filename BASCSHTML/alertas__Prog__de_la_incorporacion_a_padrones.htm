Incorporación a padrones

Mediante esta tarea es posible programar con que frecuencia se desea analizar las compras y las ventas realizadas durante los últimos doce meses.

Mediante la utilización de valores de atributos, es posible identificar a los clientes y proveedores, para evaluar si superan determinadas magnitudes, a fin de incluirlos en el padrón de tasas para hacerle retenciones o percepciones de ingresos brutos.

Programar trabajoProgramar trabajo

Para programar el la frecuencia con que se desea realizar la evaluación de los proveedores monotributistas, se debe presionar este botón. Se despliega una ventana para que se indique la frecuencia deseada.

Para programar el proceso es necesario que los campos correspondientes a atributo y valor de proveedores o atributo y valor de clientes, tengan información.

Periódico

A través de esta selección es posible programar la tarea para que se realice en base a un período determinado (días, semanas, meses).

En el primer caso es posible parametrizar el sistema para que el proceso se ejecute cada n cantidad de días.

Si se selecciona la opción semanalmente se puede parametrizar para que el proceso se ejecute cada n semanas o en determinados días de la semana.

Si se selecciona la opción mensualmente se parametriza para que el proceso se ejecute cada“n meses o determinado día de cada mes, o el primer (segundo, tercero, cuarto o último) día, por ejemplo martes, de cada mes.

Código de la empresa.

En caso de indicar un código de empresa se considerarán, para los proveedores y clientes que corresponda, sólo las operaciones realizadas en dicha empresa.

Si este campo se deja en blanco, se considerarán las operaciones realizadas por los sujetos en cada una de las empresas de la base de datos, obteniendo un total de operaciones por empresa.
