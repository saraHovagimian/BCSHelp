Etiquetas de productos

Mediante esta tarea es posible imprimir etiquetas para acompañar el envío de mercadería o simplemente para identificar a los bienes dentro del depósito.

Es posible emitir etiquetas de los bienes que están incluidos en determinados comprobantes, que están en determinados depósitos o, directamente, de aquellos que se indican al ejecutar la tarea.

Con respecto a la cantidad de etiquetas a emitir, si se solicita la emisión por producto, el sistema emitirá una etiqueta por cada uno de los bienes seleccionados. En caso que se desee se puede solicitar una etiqueta por cada color y/o medida.

Si se solicita la emisión por unidad de producto, el sistema emitirá tantas etiquetas como cantidad de bienes haya en el depósito o en el comprobante (según lo pedido). También deberá indicarse que cantidad debe tener en cuenta el sistema para la emisión, es decir si debe considerar la unidad de venta o la unidad de compra.

Los formatos pueden indicarse en Word o como archivos de formato QRP. Los detalles de variables a incluir en cada caso están disponibles en el documento Variables para informes de la documentación del producto.

Al momento de imprimir un informe (tanto por pantalla como por papel) se puede utilizar el formato standard del sistema, o indicar un formato personalizado de impresión. Este último se indica en el campo homónimo que, en general, se encuentra en la zona inferior de la primer solapa y su funcionalidad está detallada a continuación en el link Emisión.

Emisión

Los datos solicitados para la emisión de etiquetas son los siguientes:

Ficha OpcionesFicha Opciones

Ítems

En esta zona se debe dar información destinada a determinar cuáles son lso ítems para los cuáles se desea imprimir etiquetas.

En comprobante

Si se selecciona En comprobante se imprimirá una etiqueta por cada ítem incluido en cada comprobante que se indique. Las etiquetas se emitirán en el orden en que se ingresaron los productos.

En comprobante en preparación

Es posible emitir etiquetas de productos que se encuentran en comprobantes en preparación.

Para indicar los comprobantes a considerar se dispone del cuadro Comprobantes a la derecha, donde se debe indicar:

Comprobante

Fecha

Fecha de los comprobantes para los cuales se desean imprimir las etiquetas.

Podrá indicarse una fecha o dejar el dato en blanco para imprimir las etiquetas para todas las fechas.

Número

Prefijo y números de de los comprobantes para los cuales se desean imprimir las etiquetas.

Podrá indicarse un número o dejar el dato en blanco para imprimir las etiquetas para todos los números de comprobante.

Lote

Tipo y fecha del lote de los comprobantes para los cuales se desean imprimir las etiquetas.

Podrá indicarse un número o dejar el dato en blanco para imprimir las etiquetas para todos los lotes.

Carga

Número de carga de los comprobantes para los cuales se desean imprimir las etiquetas.

Podrá indicarse una carga o dejar el dato en blanco para imprimir las etiquetas para todas las cargas.

Guía

Prefijo y número de guía de los comprobantes para los cuales se desean imprimir las etiquetas.

Podrá indicarse un número o dejar el dato en blanco para imprimir las etiquetas para todas las guías.

En depósito

Si se selecciona esta opción se deberá indicar el código del depósito cuyos bienes se desea incluir en la emisión. Si se desea es posible indicar:

Ubicación

Ubicación dentro de los ítems para los cuales se desea imprimir etiquetas. Este campo sólo se habilita si se informó un depósito en el campo precedente.

Sólo ítems sin ubicación

Indicación de si se desea emitir etiquetas sólo para aquellos bienes que carecen de una ubicación informada.

Ítems seleccionados

Si se selecciona esta opción sólo se imprimirán etiquetas para los bienes que se indiquen en las fichas Ítems y Atributos de ítems.

Emitir

La información que se ingresa en esta zona está destinada a indicar la cantidad de etiquetas a emitir.

Por producto

Indicación de si la cantidad de etiquetas se indicará en base a los productos (en contraposición a la opción de indicarla en base a la unidad en la que se mide el producto.

Si la emisión corresponde al caso para ítems en comprobantes, se consolidarán las líneas del comprobante que correspondan a un mismo producto, y si corresponde, a una misma medida y color.

Detallado por talle

Si se tilda esta opción se emite una etiqueta por cada medida del producto.

Detallado por color

Si se tilda esta opción se emite una etiqueta por cada color del producto.

Cantidad de copias

Para cada etiqueta a emitir, se imprimirán tantas copias como se indique en este campo.

Lista

Si se desea incluir el precio del ítem en la etiqueta se debe indicar la lista de precios de la cual se deben tomar.

Es posible indicar una lista que se haya definido como lista dinámica.

Cuando no se indique lista de precios, o la lista de precios indicada no esté vigente a la fecha de emisión de las etiquetas o no esté autorizada, cada producto presentará precio 0 (cero).

En caso de indicar una lista de precios vigente a la fecha de emisión de las etiquetas es posible incluir la unidad de medida correspondiente a la lista de precios indicada.

Por otro lado, los productos que no pertenezcan a la lista de precios presentarán precio 0 (cero).

El precio presentará tantos decimales como se hayan indicado para los precios en los parámetros generales.

Por unidad de producto

Indicación de si se desea emitir una etiqueta por cada una de las unidades de cada bien de los seleccionados. Se debe indicar también la unidad a considerar, a saber:

Unidad de venta

Unidad de compra

Por contenedor

Indicación de si se desea que la emisión se efectúe o no normalizada a contenedor default.

Así, si por ejemplo la emisión se efectúa Por depósito y, en el mismo se dispone de 80 unidades de un bien y, si la cantidad de unidades del contenedor default es 12, se emitirán 6 etiquetas por cada contenedor de 12 unidades y 8 etiquetas para las unidades sueltas. No obstante, si la emisión se efectúa Por comprobante, se normalizará al contenedor que se haya informado en el comprobante y, en su defecto, al default.

Máximo de etiquetas por producto

Número máximo de etiquetas que se desea imprimir por producto. Así si por ejemplo, según la información suministrada corresponde emitir 35000 etiquetas de un producto y acá se indica 50, se imprimirán, solamente, 50.

Etiqueta de mas de    %

Es posible indicar un porcentaje para imprimir etiquetas de mas. Para ello las etiquetas deben emitirse por unidad de producto y sin detallar número de serie.

Ficha ÍtemFicha Ítem

Tabla de ítems

En esta se dispone de una grilla para ingresar los ítems, pudiendo también ingresar el talle y/o color si se ha seleccionado emitir las etiquetas "Por Producto" detallando por talle y/o por color.

 

Ficha Atrib. ÍtemsFicha Atrib. Ítems

Esta ficha permite filtrar los ítems por atributos y valores de atributos.

Tabla de Atributos

Esta tabla permite informar los atributos asociados a los ítems cuyas etiquetas se desea emitir con sus valores.

Atributo

Código de atributo de los ítems cuyas etiquetas se desea emitir.

De acuerdo al atributo y al valor de atributo que se indique se considerará o no cada uno de los ítems del rango seleccionado, según los atributos y valores que se hayan asignado a cada uno de ellos.

Si no se especifica ningún atributo, todos los ítems del rango solicitado intervendrán en la emisión.

Descripción

Descripción del atributo correspondiente al código informado.

Valor

Valor del atributo informado asociado a los ítems cuyas etiquetas se desea emitir.

Debe corresponder a un valor ya definido en el sistema para el atributo indicado en el campo Código.

De acuerdo al atributo y al valor de atributo que se indique se considerará o no cada uno de los ítems del rango seleccionado, según los atributos y valores que se hayan asignado a cada uno de ellos.

Si se indica un valor, se incluirán en el listado los pedidos pendientes de todos los ítems que tengan asociados el valor del atributo indicado.

Si no se especifica ningún valor de atributo, todos los ítems del rango solicitado que tengan el atributo indicado, intervendrán en la emisión.

Descripción

Descripción del valor asociado al atributo correspondiente al código informado.
