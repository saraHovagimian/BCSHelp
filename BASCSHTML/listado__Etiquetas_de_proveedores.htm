Etiquetas de proveedores

Mediante esta tarea es posible imprimir etiquetas para acompañar el envío de mercadería o para el envío de folletería, cartas y demás a los proveedores de la empresa.

La cantidad de etiquetas a imprimir depende de la opción seleccionada. Es posible imprimir las etiquetas por proveedores, por bultos o por cada comprobante que implique un ingreso de mercadería.

Si se imprimen etiquetas para recepción de mercadería se podrán seleccionar los comprobantes con los cuales se registraron esos envíos, por fecha, números de comprobante, o por lote, carga o guía. Las etiquetas que se imprimen para comprobantes podrán incluir los datos del proveedor titular de cada comprobante o de cada bulto.

Los campos a incluir en las etiquetas se detallan en el documento Variables para informes disponible en la documentación del producto.

Al momento de imprimir un informe (tanto por pantalla como por papel) se puede utilizar el formato estándar del sistema, o indicar un formato personalizado de impresión. Este último se indica en el campo homónimo que, en general, se encuentra en la zona inferior de la primer solapa y su funcionalidad está detallada a continuación en el link Emisión.

Emisión

Los datos solicitados para la emisión de etiquetas son los siguientes:

Ficha OpcionesFicha Opciones

Proveedor

Código de proveedor para el cual se desean imprimir etiquetas.

Puede seleccionarse un proveedor en particular o un grupo de ellos.

Si este dato es dejado en blanco, se imprimirán etiquetas para todos los proveedores y si se imprimen para comprobantes, se imprimirán para todos los proveedores titulares de los comprobantes que se seleccionen.

Código de selección

Permite seleccionar un conjunto de proveedores indicando un código de selección

Se podrá seleccionar alguna de las siguientes opciones, para indicar a qué deben asociarse las etiquetas que se imprimen:

Por proveedor

Si se selecciona Por Proveedor se imprimirá una etiqueta por cada proveedor del rango solicitado.

Por Remito

Por Factura

Si se selecciona Por Remito o Por Factura se podrá seleccionar el rango de comprobantes para el cual se imprimirán las etiquetas y se presentan las siguientes opciones:

Proveedor

Si se selecciona Proveedor se imprimirá la cantidad de etiquetas indicada en el campo Cantidad de copias por cada proveedor titular de los comprobantes del rango que se solicite. Si no se indica nada, se imprime una etiqueta por proveedor.

Comprobante

Si se selecciona Comprobante se imprimirá la cantidad de etiquetas indicada en el campo Cantidad de copias por cada comprobante del rango que se solicite incluyendo los datos del proveedor titular de cada uno. Si no se indica nada, se imprime una etiqueta por comprobante.

Bulto

Si se selecciona Bulto se imprimirá una etiqueta por cada bulto con los datos del proveedor titular de cada uno.

Cantidad de copias

En el caso de seleccionar Proveedor o Comprobante, es posible indicar la cantidad de copias en este campo.

Comprobante

Fecha

Fecha de los comprobantes para los cuales se desean imprimir las etiquetas.

Podrá indicarse una fecha o dejar el dato en blanco para imprimir las etiquetas para todas las fechas.

Prefijo

Prefijo de los comprobantes para los cuales se desean imprimir las etiquetas.

Número

Números de comprobante de los comprobantes para los cuales se desean imprimir las etiquetas.

Podrá indicarse un número o dejar el dato en blanco para imprimir las etiquetas para todos las números de comprobante.

Ficha ProveedoresFicha Proveedores

En esta ficha pueden indicarse diferentes filtros a aplicar a los proveedores para los cuales se emiten las etiquetas.

Pueden filtrarse los proveedores por atributos y valores de atributos.

Tabla de Atributos

Esta tabla permite informar los atributos asociados a los proveedores a los que se les desean imprimir etiquetas con sus valores.

Código

Código del atributo asociado a los proveedores a los que se les desean imprimir etiquetas.

De acuerdo al atributo y al valor de atributo que se indique se considerará o no cada uno de los proveedores del rango seleccionado, según los atributos y valores que se hayan asignado a cada uno de ellos.

Si no se especifica ningún atributo, todos los proveedores del rango solicitado intervendrán en la emisión de etiquetas.

Descripción

Descripción del atributo correspondiente al código informado.

Valor

Valor del atributo informado asociado a los proveedores a los que se les desean imprimir etiquetas.

Debe corresponder a un valor ya definido en el sistema para el atributo indicado en el campo Código.

De acuerdo al atributo y al valor de atributo que se indique se considerará o no cada uno de los proveedores del rango seleccionado, según los atributos y valores que se hayan asignado a cada uno de ellos.

Si se indica un valor, se imprimirán etiquetas para los proveedores que tengan asociados el valor del atributo indicado.

Si no se especifica ningún valor de atributo, se imprimirán etiquetas para todos los proveedores del rango solicitado que tengan el atributo indicado.

Descripción

Descripción del valor asociado al atributo correspondiente al código informado.
